My Target Users

5 February 2025

related reading:
NN/g: Problem Statement

  • Who is your user? (Be Specific)
  • What is the user’s environment?
  • Why does the user use this product?
  • How does the user feel while using this product?
My user is someone who alarms and reminders never work for. The user's environment would be screen-centered, always doing work on a screen or interacting with a screen (preferably a desk- or laptop). The user would use this product because regular alarms and reminders can be easily ignored, whereas this product would pause the screen they would be on to allow the user to leave the screen momentarily and accomplish whatever task they set a reminder for. The user feels momentarily annoyed, because their screen is no longer useable, but they would also feel A user might begrudingly do what the alarm buddy is reminding them to do, but they would be thankful in the end that they accomplished their task.
